Detailed Description

A streaming client using the Raw protocol that receives the JSONs over multiple buffers with mutexes. It applies a non-blocking round-robin approach to finding an available buffer to dump the contents in, but keeps track of sequence numbers to not lose order.

Receive JSONs into the pre-allocated buffers.

This function is meant to run in a main thread, where consuming threads access the buffers through buffer() and mutex() to absorb the data. Consuming threads must Reset() the buffers after they are done processing the data.
