Enqueue all JSONs in the TCP buffer.

Parameters
[out]json_bufferReusable JSON string buffer.
[in,out]recv_bufferThe TCP buffer that is cleared after all JSONs are queued.
[in]tcp_valid_bytesNumber of valid bytes in the TCP buffer.
[out]queueThe queue to enqueue the JSON queue items in.
[in,out]seqThe sequence number for the next JSON item, is increased when item is enqueued.
[in]receive_timePoint in time when this buffer was received.
[out]trackerLatency tracking device, set to nullptr if unused.
Returns
The number of JSONs enqueued.
