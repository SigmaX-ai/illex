Illex

A random JSON generator based on Arrow schemas.

Install

  • Requirements
    • To build:
      • CMake 3.14+
      • A C++17 compiler.
    • Dependencies:
      • Arrow 1.0.1

## Build

git clone https://github.com/teratide/illex.git
cd illex
mkdir build && cd build
cmake ..
make
make install

Usage

There are two subcommands, file and stream.

Both subcommand require an Arrow schema to be supplied as the first positional argument, or through -i or --input.

Other than that:

Option Description
-pretty to pretty-print JSON output.
--seed INT to set a seed for random generators.
-s INT same as --seed

More detailed options can be found by running:

illex --help <subcommand>

Example

After running tripreport.py in the examples folder, you could run:

illex file examples/tripreport.as -s 0 --pretty
